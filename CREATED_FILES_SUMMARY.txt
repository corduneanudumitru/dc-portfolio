QUICK START
================================================================================

1. Install dependencies:
   npm install @sanity/client @sanity/image-url sanity @sanity/structure @sanity/vision

2. Configure environment:
   cp .env.example .env.local
   # Fill in NEXT_PUBLIC_SANITY_PROJECT_ID and NEXT_PUBLIC_SANITY_DATASET

3. Deploy schema:
   npm run sanity:build

4. Access Studio:
   http://localhost:3000/studio (local)
   https://[project-id].sanity.studio (managed)

5. Create initial content:
   - Site Settings
   - About Page
   - Navigation
   - First Project/Post

6. Read documentation:
   - Start: README_SANITY.md
   - Setup: SANITY_SETUP.md
   - Usage: CMS_QUICK_START.md (for photographers)

INTEGRATION EXAMPLES
================================================================================

Importing Schemas:
  import { schemaTypes } from '@/sanity/schemas';

Fetching Projects:
  import { getAllProjects, getProjectBySlug } from '@/sanity/lib/queries';
  const projects = await getAllProjects();

Getting Site Config:
  import { getSiteSettings } from '@/sanity/lib/queries';
  const settings = await getSiteSettings();

Optimizing Images:
  import { getImageUrl, getResponsiveImageUrls } from '@/sanity/lib/image';
  const url = getImageUrl(image, { width: 800, quality: 85 });

Using Types:
  import { Project, isProject } from '@/sanity/types';
  const data: Project = await getProjectBySlug('slug');
