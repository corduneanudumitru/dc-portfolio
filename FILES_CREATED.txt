DATA FLOW
================================================================================

Sanity CMS Queries:
  ├── getAllProjects(category?)         → ProjectGrid (home page)
  ├── getFeaturedProjects()             → ProjectGrid (home page)
  ├── getProjectBySlug(slug)            → ProjectPage
  ├── getProjectSlugs()                 → generateStaticParams()
  ├── getAllPosts(category?)            → JournalPage
  ├── getPostBySlug(slug)               → BlogPostPage
  ├── getPostSlugs()                    → generateStaticParams()
  ├── getRecentPosts(limit)             → JournalScroll (home page)
  ├── getNavigation()                   → Navigation component
  ├── getSiteSettings()                 → Layout, Footer, HomePage
  ├── getAboutPage()                    → AboutPage
  ├── getProjectCategories()            → (unused in components, available)
  └── getPostCategories()               → (unused in components, available)

Form Submission:
  ContactPage → fetch(/api/contact) → POST validation → console.log()
