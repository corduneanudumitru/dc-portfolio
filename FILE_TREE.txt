================================================================================
QUERY REFERENCE
================================================================================

PROJECTS:
  getAllProjects()          → All projects (optionally filtered by category)
  getFeaturedProjects()     → Featured projects only
  getProjectBySlug(slug)    → Single project with full details
  getProjectSlugs()         → All slugs (for static generation)
  getProjectCategories()    → Unique category values
  searchProjectsByTags()    → Filter by tags
  getProjectStats()         → Count and statistics

POSTS:
  getAllPosts()             → All posts (optionally filtered by category)
  getPostBySlug(slug)       → Single post with content
  getPostSlugs()            → All slugs
  getRecentPosts(limit)     → Latest N posts
  getPostCategories()       → Unique category values
  getPostStats()            → Count and statistics

CONFIGURATION:
  getNavigation()           → Menu structure
  getSiteSettings()         → Global settings
  getAboutPage()            → Biography and exhibitions

IMAGE UTILITIES:
  getImageUrl(source, opts) → Customized image URL
  getResponsiveImageUrls()  → Mobile/tablet/desktop URLs
  getThumbnailUrl()         → Quick thumbnail
  getHeroImageUrl()         → Hero image optimization

================================================================================
GETTING STARTED FLOW
================================================================================

1. INSTALL
   └─ npm install [dependencies]

2. CONFIGURE
   └─ Create .env.local with Project ID

3. DEPLOY SCHEMA
   └─ npm run sanity:build

4. ACCESS STUDIO
   └─ https://[project-id].sanity.studio

5. CREATE CONTENT
   ├─ Site Settings (global config)
   ├─ About Page (biography)
   ├─ Navigation (menu)
   ├─ First Project (photography)
   └─ First Post (blog)

6. BUILD FRONTEND
   ├─ Import queries
   ├─ Use types for type-safety
   ├─ Optimize images
   └─ Deploy site
