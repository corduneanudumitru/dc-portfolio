# CMS Quick Start Guide for Photographers

Your photography portfolio is powered by **Sanity CMS**—a powerful content management system that lets you edit EVERYTHING from a beautiful dashboard.

## What You Can Edit

✓ **Projects** — Upload project photos, write descriptions, categorize work
✓ **Blog Posts** — Write photo essays, travel notes, gear reviews
✓ **Navigation** — Update your site menu
✓ **Site Settings** — Change your name, tagline, contact info, social links
✓ **About Page** — Your biography, portrait, equipment list, exhibitions

## Getting Started

### Step 1: Access the CMS
Go to your Sanity Studio at:
```
https://[your-studio-name].sanity.studio
```
Or if running locally:
```
http://localhost:3000/studio
```

### Step 2: Log In
Use your Sanity account credentials (the same email you signed up with).

### Step 3: Start Editing!
You'll see the main menu on the left with all sections.

## Common Tasks

### Add a New Photography Project

1. **Click "Projects"** in the left menu
2. **Click "Create"** button (top right)
3. **Fill in the required fields:**
   - Title (e.g., "Iceland Adventure 2024")
   - Category (street, travel, landscape, etc.)
   - Cover Image (upload your best shot)

4. **Add more details:**
   - Short description (shown in listings)
   - Gallery images (click to add images from your computer)
   - Project date
   - Location
   - Tags (e.g., "mountains", "north-atlantic")

5. **Optional - Go deeper:**
   - Write a full project description in the "Project Details" section
   - Toggle "Featured Project" to show on homepage
   - Set display order (lower numbers show first)

6. **Click "Publish"** when done

### Write a Blog Post

1. **Click "Blog Posts"** in the left menu
2. **Click "Create"**
3. **Fill in:**
   - Title (e.g., "5 Tips for Travel Photography")
   - Category (photo-essay, travel-notes, gear, behind-the-images)
   - Cover image
   - Excerpt (summary for listings)

4. **Write your post:**
   - Click in the "Post Content" box
   - Type or paste your text
   - Format with headings, bold, italics using the toolbar
   - Add images by clicking the image button

5. **Set publish date:**
   - Click "Published Date"
   - Choose today's date

6. **Click "Publish"**

### Update Your About Page

1. **Click "About Page"** in the left menu
2. **You can edit:**
   - Your portrait photo
   - Your name/title
   - Your full biography (with formatting)
   - Short bio for the homepage
   - Equipment list (e.g., "Canon R5", "Nikon Z9")
   - Your exhibitions and shows

3. **Click "Publish"** when done

### Change Your Navigation Menu

1. **Click "Navigation"** in the left menu
2. **Edit existing items:**
   - Change label text (e.g., "Portfolio" → "My Work")
   - Update the URL/path
   - Change the order number

3. **Add new menu items:**
   - Click the **+** button at the bottom of items
   - Enter label, URL, and order
   - Check "External Link" if it's to another website

4. **Click "Publish"**

### Update Site Settings

1. **Click "Site Settings"** in the left menu
2. **Update:**
   - Site Name (your portfolio title)
   - Tagline (e.g., "Visual Storyteller")
   - Hero Image (homepage banner)
   - Contact Email
   - Footer text (copyright notice, etc.)
   - Social media links (Instagram, Twitter, etc.)

3. **For SEO:**
   - Write an SEO Title (50-60 characters) for Google
   - Write an SEO Description (150-160 characters)

4. **Click "Publish"**

## Tips & Best Practices

### Images
- Use high-quality images (at least 2000px wide)
- The system automatically optimizes for different screen sizes
- Always add alt text for accessibility
- Use the hotspot feature to control how images crop on mobile

### Text
- Keep project descriptions under 500 words
- Use short, descriptive titles
- Bold important points for readability
- Use headings to organize longer content

### Organization
- Use consistent category names
- Tag projects to make them searchable
- Keep your portfolio fresh with recent work
- Set "Featured" on your 3-5 best projects

### SEO (Search Engines)
- Write descriptive titles (not "Project 1")
- Use meaningful keywords in descriptions
- Add alt text to all images
- Keep site name and tagline consistent

## Formatting Text

When writing project descriptions or blog posts, you can format text:

- **Bold:** Select text → click the **B** button
- *Italic:* Select text → click the *I* button
- `Code:` Select text → click the `<>` button
- # Headings: Use the dropdown menu
- Links: Select text → click the link icon → paste URL

## Managing Images

### Adding Images
1. Click the **image icon** in the editor
2. Upload from your computer or cloud storage
3. Add alt text (important for accessibility)
4. Add optional caption

### Using Hotspots
For cover images, click "Edit Hotspot":
- The hotspot shows which part of the image stays visible on mobile
- Drag the crosshair to the most important part of the photo
- This prevents important details from being cropped out

## Publishing & Scheduling

### Publish Immediately
- Click **Publish** button when done

### Save as Draft
- Click **Save** to save without publishing
- Your changes won't appear on the website until you click Publish

### Schedule for Later
- Set the date/time in the publish date field
- The content will automatically publish at that time

## Important Notes

- **Slugs are unique** — You can't have two projects with the same URL slug
- **Images load slowly?** — Wait a moment, the system is optimizing them
- **Changes not appearing?** — The website might need a few seconds to update
- **Want to unpublish?** — Click "Unpublish" button (keeps your draft)
- **Accidentally deleted?** — Check the history (look for the clock icon)

## Common Mistakes to Avoid

❌ Don't leave required fields empty (they're marked with *)
❌ Don't publish without setting a date
❌ Don't use special characters in slugs (/ ? # etc)
❌ Don't forget to publish after editing!
❌ Don't use inconsistent category names

## Need Help?

### In the Sanity Studio
- Hover over field labels for helpful descriptions
- Click the **?** icon for field information
- Use the "Question" button if something doesn't work

### Reset Your Work
- Click the **trash icon** to delete an unsaved draft
- Click **Revert** to go back to the last published version
- This won't affect your published website
